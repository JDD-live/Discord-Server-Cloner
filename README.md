# The Gods  
## Discord Server Cloner

Uma ferramenta de linha de comando simples e poderosa para clonar a estrutura completa de um servidor Discord, incluindo canais, cargos e permissões.

</div>

---

## Aviso Importante

Automatizar contas de usuário (self-bots) é contra os Termos de Serviço do Discord.  
O uso desta ferramenta é de sua total responsabilidade.

- Existe risco de suspensão ou banimento da conta.
- Recomenda-se utilizar uma conta alternativa.
- Nunca compartilhe seu token. Ele concede acesso total à conta.

---

## Funcionalidades

- Clonagem completa de categorias, canais de texto e voz, preservando a ordem original.
- Cópia de cargos com cores, ícones e permissões globais.
- Reprodução fiel das permissões específicas de cada cargo em cada canal.
- Interface interativa no terminal, sem necessidade de comandos manuais.
- Instalação automática de dependências via script.
- O token é solicitado apenas no terminal e não fica armazenado no código.

---

## Como Usar (Windows)

1. Baixe o projeto.
2. Extraia os arquivos para uma pasta local.
3. Execute o arquivo `start.bat`.
4. Siga as instruções exibidas no terminal.
5. Informe o token e os IDs dos servidores de origem e destino.

O processo de clonagem será iniciado automaticamente após a configuração.

---

## Estrutura do Projeto

- `index.js`  
  Contém toda a lógica principal do script.

- `start.bat`  
  Script que verifica dependências e executa o projeto.
